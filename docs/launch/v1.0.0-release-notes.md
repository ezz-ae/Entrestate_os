# Entrestate OS `v1.0.0` Release Notes

Release date: 2026-03-08

## Summary

Entrestate OS `v1.0.0` is the first full public platform release for disciplined real-estate decision infrastructure.

This release unifies:

- Intelligence chat and copilot surfaces
- Mind-map driven documentation and article system
- Reports library and generated report routes
- Partner, industry, careers/intern, and investor-relations information architecture
- Governance seeds for source-of-truth and investor metric auditing

## What shipped

### Platform and product surfaces

- Expanded top-level platform routes for docs, partners, APIs, industry, media, responsibility, careers, intern, and investor relations.
- Added complete docs/article routing scaffolding for the decision infrastructure map.
- Added reports library experience and generated/public report routes.

### Decision and data narratives

- Added structured decision-infrastructure article set.
- Added source-of-truth registry docs and initial data schema/csv references.
- Added investor metrics audit docs and initial data schema/csv references.
- Added a formal missing-data execution backlog for investor/partner readiness.

### Reliability hardening

- Stabilized `/api/chat` with model timeout protection and deterministic fallback behavior.
- Stabilized usage metering path with retry-safe fallback.
- Stabilized `/api/copilot` stream behavior when model/provider config is unavailable.
- Added persistence guardrails for DB schema mismatches in copilot chat history writes.

### Operational readiness

- Added Neon/Auth + LLM go-live runbook.
- Added single-file data-agent brief for notebook execution.

## Compatibility and requirements

- Next.js `16.0.10`
- Node runtime (server routes)
- Neon/Postgres database (`DATABASE_URL`, `DATABASE_URL_UNPOOLED`)
- At least one LLM provider configured (`GEMINI_KEY` or `AI_GATEWAY_API_KEY` or `OPENAI_API_KEY`)

## Known remaining gaps

Business/data completeness items remain tracked in:

- `docs/MISSING_DATA_ORDER.md`

Key unresolved launch-adjacent items include full P0 data population for KPI registry, investor audit evidence, and API contract completeness.
